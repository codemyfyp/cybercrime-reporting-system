## 📌 Project Description

The **Cybercrime Reporting System** is a secure web-based application developed using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.

This platform allows users to report cybercrime incidents online, upload evidence, track complaint status, and communicate with administrators through a secure dashboard.

It is an excellent **Final Year Project for CSE / IT / Cybersecurity Students** interested in security-focused applications and government service platforms.
